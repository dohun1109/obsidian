# 프로그램과 프로세스 
## 정적 프로그램 (Static Program)
프로그램은 원도우의 .exe 파일이나 mac 의 .dmg 파일과 같은 컴퓨터에서 실행 할 수 있는 파일을 통칭 한다. 단, 아직 파일을 실행하지 않은 상태이기 때문에 정적 프로그램 줄여서 프로그램이라고 부른 것이다. 
어떠한 프로그램을 개발하기 위해선 자바나 C언어 같은 언어를 이용해서 코드를 작성
