# HTTP 메시지에 모든 것을 전송
- HTML, TEXT
- IMAGE, 음성, 영상, 파일 
- JSON, XML(API)
- 거의 모든 형태의 데이터 전송 가능
- 서버간에 데이터를 주고 받을 때도 대부분 HTTP 사용
- 지금은 HTTP 시대 

# HTTP 역사 
- HTTP/0.9 1991년 : GET 메서드만 지원, HTTP 헤더 X 
- HTTP/1.0 1996년 : 메서드, 헤더 추가
- **HTTP/1.1 1977년 : 가장 많이 사용, 우리에게 가장 중요한 버전**
	- RFC2068(1997) -> RFC2616(1999)
